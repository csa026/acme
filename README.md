### VPS本地IP的acme证书申请一键脚本
#### 功能与特点：
1：支持纯IPV4、纯IPV6、双栈VPS

2：支持80端口模式与DNS API模式，支持单域名与泛域名

3：支持Cloudflare/腾讯DNSPod/阿里Aliyun托管解析平台的DNS API申请

4：查询、撤销并删除当前已申请的域名证书 

5：手动一键或者指定续期域名证书

#### 注意：脚本为独立模式，不建议与nginx、caddy类带有自动申请证书的应用同时使用
